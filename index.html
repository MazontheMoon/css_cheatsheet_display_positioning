What's this all about?

Previous cheatsheets such as the Box Model and Selectors have shown how the content boxes work and ways to select them, now we are moving on to where we want to put the content boxes in our document.

Display

Everything is a box and the type of box applied to an element is specified by the display property.

  • Block
  • Inline
  • Inline-Block
  • Flex
  • Grid
  • None

Positioning

Display positions the box for the element in the body of the document and position is used to specify where an element is positioned within it's container box.

Position values are:

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky

The elements are then positioned using top, right, bottom and left values (can also use clip)
